Sun’îmakam taksim · fretless guitarv3.20

Sun’î improvises taksims over the 13 basic makams of Turkish music. The generation is random, yet the seyir (melodic direction) and the pull toward the güçlü (dominant) and karar (final) are fixed, so every run is different but recognizably the same makam. Pitches come from the AEU 53-comma system and are voiced with true microtones. Try Hicaz, then Uzzal: identical scale, different dominant. Name a piece, Save it as a small file, and Open it later to replay it exactly.

Green = güçlü (dominant / half-cadence pull)  ·  Red = karar (final / tonic)  ·  Yellow = perde currently sounding. Comma values are shown between perdes.
Engine: seyir-aware weighted random walk. Phrase endings lock onto asma karar (secondary-cadence) perdes; the final phrase closes with the yeden → durak (leading-tone → tonic) formula. Save stores the exact note sequence, so Open + Play reproduces a piece identically. The voice is a sampled acoustic guitar, pitch-shifted to the exact makam commas, so it plays true microtones (fretless). Guitar samples: tonejs-instruments (CC-BY 3.0).
